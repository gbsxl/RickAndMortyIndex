# Rick and Morty - API Project

## O que é

Um site que mostra personagens da série Rick and Morty. Os dados vêm de uma API pública.

## Como funciona

1. Quando você abre a página, o JavaScript faz uma requisição para a API
2. A API retorna os dados dos personagens
3. O JavaScript cria cards dinamicamente com as informações
4. Os cards aparecem na tela organizados em grid

## Arquivos

- index.html - Estrutura da página
- style.css - Estilos do site
- script.js - Código que faz o fetch e cria os cards

## Como abrir

Abra o arquivo index.html em um navegador web. Pronto, os cards vão carregar automaticamente.

## Requisitos

- Um navegador moderno (Chrome, Firefox, Safari, Edge)
- Conexão com internet

## API utilizada

Rick and Morty API - https://rickandmortyapi.com/
Não requer autenticação.
